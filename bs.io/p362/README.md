# Odd Palindrome

This is problem #362 found at https://binarysearch.com/problems/Odd-Palindrome

## Statement

Given a string `s`, return whether all its palindromic substrings have odd lengths.

### Examples

**Input**
```
s = 'faf'
```
**Output**
```
true
```
**Input**
```
s = 'faab'
```
**Output**
```
false
```
